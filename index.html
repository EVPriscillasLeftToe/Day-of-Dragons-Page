The game

Day of Dragons is an online creature survival game set in a large, beautiful open world with distinct creatures. You can play as one of several dragon species or (in the future) as an elemental.
Please note: this game is still in Early Access, bugs and glitches can be expected.

Updates

Since the game is still in Early Access, updates and patches will be dropped frequently. Some may drop earlier than others as the creators of this game prefer quality over quantity.

Some of the following updates will be:

  • 1.3 This update will bring the new skin crafting system allowing players to create their own skins. Players will also be able to tame their own pets. It is said to drop somewhere in 2026.
  • 1.4 This update will add a new species called "Singe Crest" (a fish-eating wyvern) along with the whiskered paddletail (NPC fish).
  • 1.5 The new species "Zygovo" (a tiny egg-eating dragon) will be added.
  • 1.x More updates will be coming, adding much more content and creatures.
